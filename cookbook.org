* File

** Read File

As a string.

#+BEGIN_SRC lisp
  (with-open-file (stream "~/src/lisproom/README.org")
    (let ((contents (make-string (file-length stream))))
      (read-sequence contents stream)
      contents))
#+END_SRC

As a list of lines.

#+BEGIN_SRC lisp
  (with-open-file (stream "~/src/lisproom/README.org")
    (loop for line = (read-line stream nil)
       while line
       collect line))
#+END_SRC
